On GitHub:

Create a new repository — name it coordinate-explorer (make it Public)
Click "Add file" → "Upload files" and drag in the index.html
Click "Commit changes"
Go to Settings → Pages → set Source to "Deploy from a branch", branch = main, folder = / (root) → Save
Wait about 60 seconds — your site will be live at:
https://yourusername.github.io/coordinate-explorer/
